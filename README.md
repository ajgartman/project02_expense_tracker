CLI Expense Tracker
Tier 1 / Project 02

Hello! Here is my project 02. 

Description:
A command-line expense tracker built to practice JSON, data structures, and filtering in Python.
Expenses are stored locally in a JSON file and can be summed, filtered by category, and sorted by date, 
with category filtering powered by pandas.

Features:

1. Add an expense with category, amount, date, and location
2. View all logged expenses
3. Sum the total amount spent
4. Filter expenses by category (Food, Transport, Utilities, Entertainment)
5. View expenses sorted by most recent date
6. Look up expenses for a specific date
